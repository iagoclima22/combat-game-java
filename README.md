# Champion Battle – Java OOP

Projeto simples em Java que simula uma batalha entre dois campeões. Cada um possui vida, ataque e armadura. A cada turno, ambos causam dano e o status é exibido até que um deles chegue a 0 de vida.

## Conceitos usados
- Classes, objetos e construtores
- Encapsulamento básico
- Métodos para cálculo de dano e exibição de status
